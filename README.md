- Kali Linux
- Social Engineering Toolkit (SET)

### Configuração do Phishing
1. Acesso root:  
   ```sudo su```  
2. Iniciar setoolkit:  
   ```setoolkit```  
3. Opções selecionadas:

   - Vetor de ataque: **Web Site Attack Vectors** (opção 2)  
   - Método de ataque: **Credential Harvester Attack Method** (opção 3)
